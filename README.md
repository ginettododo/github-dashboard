# RepoRadar (Scaffold)

## Install
```bash
npm install
```

## Run in development
```bash
npm run dev
```

## Build renderer + electron outputs
```bash
npm run build
```

## Package installers/artifacts
```bash
npm run dist
```

### Windows artifacts
```bash
npm run dist:win
```

### macOS artifacts
```bash
npm run dist:mac
```

> Note: Cross-platform packaging may require running on the target OS (or extra toolchains/signing setup).
